Node List

In this practice case, you will play with HTML objects that permit multiple selection. It introduces you to having to use the querySelectorAll() method to access user choices and to processing the resulting NodeList — a slice of nodes from the DOM.

In this exercise, we will build a wee section of an order system for paintball equipment!

Exercise Challenges:

  • Make a HTML webform with a select object that implements a multiples-multiple" attribute with the next items:
    • Valken Green Gas - 8oz Can
    • Tango Mesh Mask - For Airsoft
    • Condor Elbow Pad
    • SpeedQB Atom Backpack
  • Prices for the items, above, are as follows:
    • Green Gas: $15.99
    • Mesh Mask: $22.99
    • Elbow Pad: $39.99
    • Backpack: $239.99
  • Here are some business & progrartrning ruies/suggestjons foc processing the order:
    • The customer will receive a 15% discount if s/he orders 3 items and a 20% discount if s/he orders all 4 items. No discount savings are given for 1 or 2 items purchased.
    • The order total is the sum of the selected item prices — the discount
    • Remember to use functions to organize your processing!
  • Use an alert to output the order total. Inform the user of the discount for the order.
