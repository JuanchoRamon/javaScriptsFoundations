Body Mass Indicator (BMI)

It is a simple measurement used to estimate whether a person’s weight is appropriate for their height, and it is calculated as follows: weight/height2

Lab Challenge:

For now, we will code this calculator magically assuming all users enter valid weight and height data in metric (kilograms for weight and meters for height).

In this assessment, you are asked to:

  • assemble a typical web project, with an index page and a scripts folder, which will contain a .js file. Name your project folder, YourName_Lab1l
  • using HTML, design a web form that gathers the input required to calculate BMI and invoke a click event of a button
  • using JS, code the button’s click event handler to calculate the BMI result, given the user input, and output the BMI outcome to an alert (pop-up)
