Guess the Number

In this assessment you will create a web interface that allows the user to generate a number between:

  • 1-10
  • 1-50
  • 1-100

Once a range is selected, the user will guess a number between that range. Validate that the number falls within the range. Do not process the guess until you have a valid input from the user and output an error message if it is invalid. If guess is valid, and the user guesses correctly, output a congratulations message; otherwise, inform the user of the miss and what the random number was.

Lab Challenge:

  • Use one function to generate the random number between the chosen range
  • Use another function to validate the range
